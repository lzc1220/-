## 安装与运行

项目运行之前，请确保系统已经安装以下应用

- node (6.0 及以上版本)

### 运行命令

```
git clone git clone https://github.com/lzc1220/-.git
cd final/fe

npm install

#development
npm run start

#production
npm run build
```

### 关于生产环境

前端打包到后端，并起一个static server，**请用```${后端的主机+端口}/index.html```访问**
